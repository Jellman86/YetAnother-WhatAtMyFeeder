# Support

Need help with YA-WAMF? Here are a few ways to get support:

1.  **Check the Documentation:** Read the `README.md` and check the Wiki (if available) for configuration guides.
2.  **Search Issues:** Someone else might have had the same problem. Search the [GitHub Issues](https://github.com/Jellman86/YetAnother-WhosAtMyFeeder/issues).
3.  **Open an Issue:** If you can't find a solution, open a new issue. Please provide:
    *   Your Docker Compose configuration (redact passwords!).
    *   Logs (`docker compose logs`).
    *   Browser info.

**Note:** This is an open-source project maintained by volunteers. Response times may vary.
